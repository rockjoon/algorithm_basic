# 1048 한 번에 2의 거듭제곱 배로 출력하기



## 문제

정수 2개(a, b)를 입력받아 a를 2 ** b배 곱한 값으로 출력해보자.
0 <= a <= 10, 0 <= b <= 10




---
### 입력 

정수 2개가 공백을 두고 입력된다.   
0 <= a, b <= 10



---
### 출력   

a 를 2**b배 만큼 곱한 값을 출력한다.


---
### 입력 예시

1 3

---
### 출력 예시

8

---
제출
---
```python
a,b=input().split()
a=int(a)
b=int(b)
print(a * (2 ** b))
```
---
### 풀이
* 문제를 꼼꼼히 읽자!(나는 당연히 문제가 a의 b제곱을 구하라는 것일 줄 알았는데 아니었다...)
* 파이썬에서 제곱을 구하기 위해서는 `**` 연산자를 사용
