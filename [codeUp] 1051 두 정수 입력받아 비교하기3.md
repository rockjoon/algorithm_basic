# [codeUp] 1051 두 정수 입력받아 비교하기3 파이썬 풀이



## 문제

두 정수(a, b)를 입력받아   
b가 a보다 크거나 같으면 1을, 그렇지 않으면 0을 출력하는 프로그램을 작성해보자.




---
### 입력 

정수 2개가 공백을 두고 입력된다.   
-2147483648 <= a, b <= +2147483647



---
### 출력   

b가 a보다 크거나 같은 경우 1을, 그렇지 않은 경우 0을 출력한다.


---
### 입력 예시

0 -1
---
### 출력 예시

0

---
제출
---
```python
a,b=input().split()
a=int(a)
b=int(b)
z=int(a<=b)
print(z)
```
---
### 풀이
* 저번 문제와 마찬가지로 if문을 쓰지 않는게 더 효율적이다.
