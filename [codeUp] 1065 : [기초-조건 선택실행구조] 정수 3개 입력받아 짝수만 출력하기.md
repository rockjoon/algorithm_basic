# [codeUp] 1065 : [기초-조건/선택실행구조] 정수 3개 입력받아 짝수만 출력하기 python


## 문제
 
세 정수 a, b, c가 입력되었을 때, 짝수만 출력해보자.





---
### 입력 

3개의 정수가 공백을 두고 입력된다.
-2147483648 ~ +2147483647


---
### 출력   

짝수만 순서대로 줄을 바꿔 출력한다.

---
### 입력 예시

1 2 4

---
### 출력 예시

2
4

---
제출
---
```python
a,b,c=map(int, input().split())
if a % 2 == 0:
    print(a)
if b % 2 == 0:
    print(b)
if c % 2 == 0:
    print(c)
```

---
### 풀이
* 음, 답이 예쁘진 않다.
