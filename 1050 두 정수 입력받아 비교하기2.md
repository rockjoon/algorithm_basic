# 1050 두 정수 입력받아 비교하기2



## 문제

두 정수(a, b)를 입력받아  

a와 b가 같으면 1을, 같지 않으면 0을 출력하는 프로그램을 작성해보자.




---
### 입력 

정수 2개가 공백을 두고 입력된다.   
-2147483648 <= a, b <= +2147483647



---
### 출력   

a와 b의 값이 같은 경우 1을, 그렇지 않은 경우 0을 출력한다.


---
### 입력 예시

0 0

---
### 출력 예시

1

---
제출
---
```python
a,b=input().split()

x=int(a)
y=int(b)
z=int(x==y)

print(z)
```
---
### 풀이
* if문으로 a,b가 같은지 비교하여 출력할 수 도 있지만,
true/false를 int로 변환하여 출력하는 것이 더 간단하다.
